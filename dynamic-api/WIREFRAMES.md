# Mister DJ Website - Wireframes

## Design System Summary

### Colors
- **Primary Brand**: Blue gradient (`brand-gradient`)
- **Secondary**: Gold/accent
- **Neutral Dark**: Text
- **Neutral Light**: Backgrounds
- **Success**: Green (validation)
- **Destructive**: Red (errors)

### Typography
- **Display Font**: Montserrat (headings)
- **Body Font**: Inter (paragraphs)
- **Sizes**: h1, h2, h3, body, small

### Spacing
- **System**: xs, sm, md, lg, xl, 2xl, 3xl
- **Container**: max-width with padding

## Component Mapping

### Components to Use:

From `/components/organisms/`:
- **HeroSection** (homepage, service pages)
- **PricingTables** (pricing section)
- **ContentHubShowcase** (gallery preview)
- **PersonaMatchShowcase** (testimonials with persona matching)

From `/components/molecules/`:
- **Forms** (contact forms with validation)
- **Cards** (service cards, team cards)

From `/components/atoms/`:
- **Buttons** (CTAs throughout)
- **Inputs** (form fields)
- **Typography** (headings, paragraphs)

From `/components/ui/`:
- **Button** (primary CTAs)
- **Card** (service cards)
- **Input, Textarea, Select** (contact forms)
- **Checkbox, Switch** (form options)
- **Alert** (form validation messages)
- **Avatar** (testimonials)
- **Accordion** (FAQ sections)
- **Tabs** (gallery categories)

From `/components/patterns/`:
- **ABTestHeroExample** (A/B test hero variants)
- **OptimizedVideo** (showreel videos)

## Interaction Patterns

### Buttons
- **Primary**: Filled, brand color, for main CTAs
- **Secondary**: Outlined, for secondary actions
- **Ghost**: Transparent, for tertiary actions
- **Hover**: Slight scale (1.02), shadow increase

### Cards
- **Hover**: Lift effect (shadow increase)
- **Click**: Navigate or modal open

### Forms
- **Focus**: Blue ring around input
- **Validation**: Real-time (on blur)
- **Error**: Red border + error message below
- **Success**: Green border + success message

### Gallery
- **Image Hover**: Overlay with title + category
- **Click**: Open lightbox/modal
- **Lightbox**: Keyboard navigation (arrows), ESC to close

### Testimonials
- **Desktop**: 3 column grid
- **Mobile**: Swipeable carousel

### FAQ
- **Accordion**: Click to expand/collapse
- **Icon**: ▶ (collapsed) ▼ (expanded)

### Video
- **Autoplay**: Muted, on hero sections
- **User-initiated**: With controls, in gallery
